Bienvenidos

HTML (lenguaje de marcado de hipertexto), CSS (hojas de estilo en cascada) y JavaScript (JS) son las tecnologías básicas para crear páginas web. HTML proporciona la estructura del contenido, CSS la presentación y JavaScript la interactividad. El DOM (modelo de objetos del documento) es la representación en árbol de la estructura de la página que permite a JS acceder dinámicamente a los elementos. Git es un sistema de control de versiones que se utiliza para gestionar cambios en el código, colaborar con otros y gestionar eficazmente múltiples versiones de un proyecto.

Preguntas sobre HTML (HyperText Markup Language)

1) ¿Qué es HTML y cuál es su función principal en el desarrollo web?

+

HTML (HyperText Markup Language) es el lenguaje de marcado de hipertexto que se utiliza para crear y estructurar páginas web. Es el estándar de facto para la codificación de páginas web y se utiliza en la mayoría de los navegadores.

Función principal:

  • Proporciona la estructura y el contenido de una página web.

  • Permite a los desarrolladores web crear páginas web con texto, imágenes, enlaces, formularios y otros elementos.

  • Facilita la accesibilidad y la indexación de las páginas web por parte de los motores de búsqueda.

2) ¿Qué son las etiquetas HTML y cómo se utilizan?

+

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo. Es decir, las etiquetas HTML son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML.

Cómo se utilizan

  • Se escriben entre corchetes angulares < >

  • Generalmente ocurren en pares: una etiqueta de apertura y una etiqueta de cierre.´00

  • La mayoría de etiquetas deben cerrarse como se abren, pero con una pleca '/'

3) Proporciona ejemplos de algunas de las etiquetas HTML más comunes.

+

Algunos de los ejemplos de etiquetas más comunes en HTML son:

  • <p> Indica el inicio de un párrafo

  • </p> Indica el final de un párrafo

  • <a> Crea un enlace a una página web

  • <img> Inserta una imagen en la página

  • <h1> a <h6> Crea encabezados de diferentes tamaños

  • <ul> Crea una lista desordenada

  • <ol> Crea una lista ordenada

  • <li> Crea un elemento de lista

  • <table> Crea una tabla

  • <tr> Crea una fila en una tabla

  • <td> Crea una celda en una tabla

  • <strong> Dá enfasis a una parte del texto

  • <em> Crea un texto en cursiva

  • <form> Crea un formulario

Otros usos de etiquetas en HTML son:

  • <header> División estructural para una cabecera de la página

  • <footer> División estructural para un pie de página

  • <nav> División estructural para una barra de navegación

  • <section> División estructural para una sección de la página

  • <article> División estructural para un artículo de la página

  • <aside> División estructural para contenido relacionado

  • <div> División genérica para agrupar elementos

  • <script> Código de un script integrado en la página

  • <head> Indica la parte superior del documento HTML
