Bienvenidos

HTML (lenguaje de marcado de hipertexto), CSS (hojas de estilo en cascada) y JavaScript (JS) son las tecnologías básicas para crear páginas web. HTML proporciona la estructura del contenido, CSS la presentación y JavaScript la interactividad. El DOM (modelo de objetos del documento) es la representación en árbol de la estructura de la página que permite a JS acceder dinámicamente a los elementos. Git es un sistema de control de versiones que se utiliza para gestionar cambios en el código, colaborar con otros y gestionar eficazmente múltiples versiones de un proyecto.

Preguntas sobre CSS (Cascading Style Sheets)

4) ¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

+

El CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML. Es decir, permite generar el diseño visual de páginas web e interfaces de usuario.

CSS (Cascading Style Sheets - en español Hojas de Estilo en Cascadas) es usado para darle estilo y diseño a las páginas Web — por ejemplo, para cambiar la fuente de letra, color, tamaño y el espaciado del contenido; dividir en múltiples columnas, o agregar animaciones y otras propiedades decorativas.

5) ¿Cuáles son las propiedades más importantes de CSS?

+

Las propiedades CSS más importantes son las que permiten definir el estilo y el comportamiento de los elementos HTML. Entre ellas se encuentran las propiedades de texto, las propiedades de imagen, las propiedades de ancho y alto, y las propiedades de borde.

Propiedades de texto:

  • Color (color), Fuente (font-family), Tamaño de fuente (font-size), Alineación (text-alignment), Decoración (text-decoration), Espaciado (text-spacing-trim).

Propiedades de imagen:

  • Ancho (width), Alto (height), Posición (position), Fondo (background-color), Opacidad (opacity).

Propiedades de borde:

  • Color (border-color), Grosor (border-width), Estilo (border-style), Radio de borde (border-radius).

Propiedades de ancho y alto

  • Ancho (width), Alto (height), Márgenes (margin), Relleno (padding).

6) Explica el modelo de caja (box model) en CSS.

+

El modelo de caja CSS es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual.

El modelo de caja CSS se compone de cuatro partes: el contenido, el relleno, el borde y el margen. Cada una de estas partes se representa como una caja rectangular, y se apilan una encima de la otra para formar la caja completa.

7) ¿Qué es el diseño responsivo y cómo se logra con CSS?

+

El diseño responsivo es una técnica de diseño web que permite que las páginas web se adapten automáticamente al tamaño de la pantalla del dispositivo en el que se están visualizando. Esto se logra mediante el uso de CSS y HTML para crear un diseño flexible y adaptable que se ajuste a diferentes tamaños de pantalla y dispositivos.

Para lograr un diseño responsivo con CSS, se utilizan técnicas como el uso de unidades de medida relativas (por ejemplo, porcentajes en lugar de píxeles), el diseño en cuadrícula (grid layout), y los medios de consulta (media queries) para aplicar estilos específicos a diferentes tamaños de pantalla.
