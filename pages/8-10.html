Bienvenidos

HTML (lenguaje de marcado de hipertexto), CSS (hojas de estilo en cascada) y JavaScript (JS) son las tecnologías básicas para crear páginas web. HTML proporciona la estructura del contenido, CSS la presentación y JavaScript la interactividad. El DOM (modelo de objetos del documento) es la representación en árbol de la estructura de la página que permite a JS acceder dinámicamente a los elementos. Git es un sistema de control de versiones que se utiliza para gestionar cambios en el código, colaborar con otros y gestionar eficazmente múltiples versiones de un proyecto.

Preguntas sobre JavaScript y DOM (Document Object Model)

8) ¿Qué es JavaScript y cómo se integra en el desarrollo web?

+

JavaScript es un lenguaje de programación de alto nivel que se utiliza para crear interactividad en las páginas web. Se integra en el desarrollo web mediante la inclusión de scripts en el código HTML o a través de archivos externos. JavaScript puede interactuar con el HTML y el CSS a través del DOM (Document Object Model), permitiendo la manipulación dinámica de los elementos de la página.

9) Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model).

+

JavaScript interactúa con el HTML y el CSS a través del DOM (Document Object Model) de la página web. El DOM es una representación en árbol de la estructura de la página que permite a JavaScript acceder y manipular dinámicamente los elementos de la página. JavaScript puede modificar el contenido, los estilos y los atributos de los elementos HTML y CSS mediante la manipulación del DOM, lo que permite crear páginas web interactivas y dinámicas.

10) Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.

+

JavaScript puede hacer que una página web sea interactiva mediante la manipulación del DOM y la interacción con el usuario. Por medio de eventos, atributos y propiedades.

  • Mostrar u ocultar elementos de la página.

  • Modificar el contenido de un elemento.

  • Validar formularios antes de enviarlos.

  • Crear efectos de animación y transición.

  • Responder a eventos del usuario, como clics, desplazamientos y teclas.
